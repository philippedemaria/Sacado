    Avant de lancer votre fichier csv, assurez-vous que tous les élèves ont été supprimés de la base de données.
    Votre fichier csv doit être encodé en utf-8 et les données doivent vérifier le format imposé (comme exemple). Ces données doivent être séparées d'un point virgule (;). Cinq champs sont demandés :
    • Classe
    • Code du niveau Noter le code du niveau : CP = 1 | CE1 = 2 | CE2 = 3 ..... 6ème = 6 | 5ème = 7 | ... 2nde = 10 | Term = 12
    • Nom
    • Prénom
    • Courriel, optionnel mais préférable pour la correspondance et les notifications.
    Ces cinq données s'écrivent sur une seule ligne. Dans l'exemple ci-dessous, 4 personnes sont enregistrées.
    4ème1;8;Alan;alan.turing@enigma.edu 
    4ème1;8;Agnesi;Maria
    4ème2;8;Chasles;Michel;michel.chasles@vector.edu
